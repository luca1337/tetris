requirements:

    VSCode Extensions:
        -C/C++ Extension
        -Cmake Tools
        -Cmake Language support
        
    compilers:
        - GCC (x86 or x64)
        - Clang (either x86 or x64)
        - MSVC (visual studio default compiler)

    other tools:
        - GDB (Gnu Debugger for using breakpoints)
        - Ninja (for cmake)
        - Cmake and Make ()

Note: windows users can use MSYS2 to install all the packets easly!
